WeatherDashboard

Github: https://github.com/timothymickiewicz/WeatherDashboard

Live-Link: https://timothymickiewicz.github.io/WeatherDashboard/

A weather dashboard to show you the current weather and 5 day forecast for any city that the user enters. Past searched cities are stored and can be revisited. The last searched city is pulled up on page load. Information is pulled via AJAX, and applied to the page with jQuery.

BranchManager

Github: https://github.com/timothymickiewicz/BranchManager

A node CLI application to create an html document with all of your team in it. You can access all of their information at a quick glance, and see information such as: Name, Email, Role, School (for interns), Office Number (for managers), and GitHub username (for engineers). This application was created using: HTML5, CSS3, JavaScript, and NodeJS as well as jest and inquirer from the npm library.
